# Заказы поставщиков — десктопное приложение для учёта закупок

Десктопное приложение на Python (tkinter) для ведения учёта заказов поставщиков, отслеживания оплат и формирования Excel‑отчётов.  
Реализованы справочники товаров и поставщиков, полный CRUD заказов с товарными позициями, а также автоматическое создание круговых диаграмм в отчётах.

## 🚀 Запуск

### Локальный запуск (Windows / Linux / macOS)

```bash
# Клонируйте репозиторий
git clone <your-repo-url>
cd order-supplier

# (Опционально) Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate      # Linux/macOS
# или
.\venv\Scripts\activate       # Windows

# Установка зависимостей
pip install psycopg2-binary openpyxl sqlalchemy

# Настройте базу данных (см. ниже)
# Запуск приложения
python main.py

Примечание: PostgreSQL должен быть установлен и запущен. Создайте базу данных nes и укажите свои параметры подключения в db.py.

### Форма заказа

- Выбор поставщика из выпадающего списка.
- Поля ввода дат (формат `YYYY-MM-DD`).
- Статус заказа.
- Блок добавления товаров:
  - Выбор товара из справочника.
  - Автоподстановка цены.
  - Ввод количества → автоматический расчёт суммы позиции.
  - Кнопки «Добавить» / «Удалить выбранный».
- Динамическое отображение общей суммы заказа.
- Кнопки «Сохранить» и «Отмена».

### Форма оплаты

- Ввод суммы, даты, способа оплаты (наличные / безналичный расчёт).
- После сохранения сумма оплаты добавляется к заказу.

### Справочники

- **Товары:** список с возможностью добавления/редактирования/удаления (название, единица измерения, цена, описание).
- **Поставщики:** добавление нового (имя, контакт, телефон, email, адрес).
