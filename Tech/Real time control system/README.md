# Система управления МНЛЗ — моделирование уровня металла в кристаллизаторах

Десктопное приложение на Python (tkinter) для моделирования и визуализации системы автоматического управления уровнем металла в четырёх ручьях машины непрерывного литья заготовок (МНЛЗ).  
Реализована физическая модель изменения уровня, двухпозиционный релейный регулятор с зоной нечувствительности 2% и запретом по знаку производной(алгоритм регулирования скорости вытягивания заготовки через изменение напряжения на приводе ролика)  и ручное вмешательство.

## 🚀 Запуск

### Локальный запуск (Windows / Linux / macOS)

```bash
# Клонируйте репозиторий (или сохраните файл m_n_lz.py)
git clone <your-repo-url>
cd mnsl-control

# Создайте виртуальное окружение
python -m venv venv
source venv/bin/activate      # Linux/macOS
# или
.\venv\Scripts\activate       # Windows

# Установка зависимостей
# tkinter обычно уже установлен, но для Linux может понадобиться:
# sudo apt install python3-tk

# Запуск приложения
python m_n_lz.py
```
## ⚙️ Параметры моделирования (настройки)

Все параметры заданы в коде внутри класса `MNSLControlSystem` и могут быть изменены вручную:

| Параметр | Значение по умолчанию | Описание |
|----------|----------------------|----------|
| `ADC_resolution` | 10 бит | Разрешение АЦП (измерение напряжения ДУМК) |
| `DAC_resolution` | 16 бит | Разрешение ЦАП (управление напряжением) |
| `ADC_input_range` | `(0, 64)` В | Диапазон входного напряжения АЦП |
| `DAC_output_range` | `(0, 24)` В | Диапазон выходного напряжения ЦАП |
| `target_level` | 70 % | Целевой уровень металла в кристаллизаторе |
| `control_step` | 0.025 В | Шаг изменения напряжения при коррекции |
| `control_interval` | 0.2 с | Интервал срабатывания регулятора |
| `level_threshold` | 2 % | Отклонение, при котором начинается регулирование |
| `level_0_percent_voltage` | 50 В | Напряжение ДУМК при 0% уровня |
| `level_100_percent_voltage` | 10.5 В | Напряжение ДУМК при 100% уровня |
